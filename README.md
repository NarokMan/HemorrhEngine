The editor designed to help make my new hockey game project.
I've already made quite a bit of progress outside of git, 
but I'm choosing to upload now so I can work on it away from
home.

Just gonna say this is like version 0.1.

CURRENT FEATURES:
- Grid workspace
  - Highlighted selected cluster
- Toolbar (on the left)
  - Creation of collision clusters
  - Creation of trigger clusters
  - Creation of collison nodes
  - Creation of trigger nodes
  - Deletion of collision clusters
  - Deletion of trigger clusters
- Action bar (up top)
  - Save maps to .cfg fomrat along with map folders
 
PLANNED FEATURES:
- Loading maps
- Running maps
- Zooming in/out on workspace
- Panning on workspace
- Select map music
- Select play starting position
